# Read-My-Lips
Read My lips is an online game for friends and Family which allows a player to guess what a person is whispering
## Purpose
To entertain and keep boredom away 

## Built With
* HTML  * 2 APIs(Giphy and Jokes)
* CSS
* JS
* Bulma
* Figma
* J Query
